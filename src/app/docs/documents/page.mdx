# Documents

The Documents section is where you manage all yacht-related files -- contracts, certificates, insurance papers, and more.

![Documents page](/images/screenshots/documents-list.png)

## Overview

The Documents page shows all documents you have access to. Each document card displays:

- **Document title** -- The name of the document (e.g., "Charter Agreement -- Oceane")
- **Language tag** -- The language of the document (e.g., EN for English)
- **Company and yacht** -- Which company and yacht the document belongs to
- **Upload date** -- When the document was uploaded
- **Download button** -- Click to download the document file

## Document Categories

Documents are organized into categories using tabs at the top of the page:

| Tab | Description |
|---|---|
| **All Documents** | Shows every document you have access to |
| **Boat** | Boat-specific documentation |
| **Contracts** | Charter agreements and contracts |
| **Insurance** | Insurance certificates and policies |
| **Transport** | Transport-related documents |
| **Legal** | Legal documentation |
| **Finance** | Financial documents |
| **Other** | Documents that don't fit other categories |

Click a tab to filter documents by category.

## Quick Actions

- **Upload Document** -- Click the button in the top-right to upload a new document (see [Upload a Document](/docs/documents/upload-document))
- **Search** -- Type in the search bar to find documents by title, filename, or tags
- **Filter** -- Use the filter button to narrow results by company or yacht (see [Filters & Search](/docs/documents/filters-and-search))
- **Download** -- Click the download icon on any document card to save the file
